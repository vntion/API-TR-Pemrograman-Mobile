Kamu adalah seorang AI Assistant yang ahli dalam pengembangan Next.js (App Router) dan TypeScript. Tugasmu adalah membuatkan fitur halaman dokumentasi API interaktif untuk proyek saya.

Untuk mempermudah pembuatan dokumentasi dan agar sesuai dengan standar OpenAPI, gunakan library next-swagger-doc dan swagger-ui-react.

Tolong berikan panduan langkah demi langkah beserta kode lengkapnya untuk:

Instalasi: Perintah untuk menginstal dependency yang dibutuhkan (next-swagger-doc, swagger-ui-react, dan tipe bawaannya).

Konfigurasi Swagger: Buat file utilitas atau konfigurasi untuk mendefinisikan spesifikasi awal OpenAPI (info, versi, komponen security jika ada).

Halaman UI Dokumentasi: Buat file app/api-docs/page.tsx yang me-render komponen dari swagger-ui-react untuk menampilkan UI dokumentasi API secara interaktif.

Contoh Endpoint dengan Anotasi: Buat satu contoh file API Route sederhana (misalnya app/api/users/route.ts) lengkap dengan anotasi komentar JSDoc/YAML standar yang dapat dibaca oleh next-swagger-doc untuk mendeskripsikan request dan response-nya.

Pastikan semua kode menggunakan TypeScript, ditulis dengan clean code, dan kompatibel dengan arsitektur Next.js App Router. Berikan penjelasan singkat pada bagian-bagian krusial agar saya mudah memahaminya.
